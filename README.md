# healthcare-kpi-dashboard
A Power BI dashboard analyzing healthcare KPIs including patient flow, wait times, appointment volume, and operational efficiency. Built using synthetic clinical data to demonstrate healthcare analytics and data storytelling.

# Project Overview
This project demonstrates my ability to transform clinical and operational data into clear, accessible insights for healthcare leadership. The dashboard highlights key performance indicators (KPIs) related to patient flow, appointment volume, wait times, and operational efficiency.

The goal was to create a communication‑friendly dashboard that supports decision‑making and helps clinical teams understand performance trends at a glance.

**Objectives**
-Visualize core healthcare KPIs in a clear, intuitive format
-Identify operational bottlenecks and trends
-Improve communication between clinical and administrative teams
-Provide leadership with actionable insights for staffing and scheduling decisions

**Tools & Skills Used**
-Power BI (data modeling, DAX, dashboard design)
-SQL (data cleaning + validation)
-Excel (pre‑processing)
-Data storytelling
-Healthcare operations knowledge
-Communication through visuals

**Dashboard Features**
-Patient volume trends (daily/weekly/monthly)
-Wait time analysis
-Appointment type breakdown
-No‑show & cancellation rates
-Operational bottleneck identification
-Clinician utilization metrics

# Approach

**1. Data Cleaning & Validation**
-Removed duplicates and inconsistent entries
-Standardized appointment types and clinician names
-Validated timestamps and corrected missing values

**2. Data Modeling**
-Built relationships between patient, appointment, and clinician tables
-Created DAX measures for KPIs (wait times, utilization, volume trends)

**3. Dashboard Design**
-Designed a clean, intuitive layout
-Used color coding to highlight performance thresholds
-Ensured accessibility for non‑technical clinical leaders

**Key Insights**
-Identified peak appointment times and staffing gaps
-Highlighted wait time spikes linked to specific appointment types
-Revealed under‑utilized clinician hours
-Provided actionable recommendations for scheduling improvements

Healthcare KPI reporting • Data storytelling • Dashboard design • Healthcare operations • Communication clarity
