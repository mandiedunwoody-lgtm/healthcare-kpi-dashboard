**Clinical Workflow Documentation & Communication Guide**

Created a clear, structured documentation guide for clinical staff outlining EMR workflows, data accuracy standards, and communication protocols. Improved consistency, reduced documentation errors, and supported onboarding of new staff.

**Project Overview**

This project demonstrates my ability to translate clinical workflows into clear, structured documentation that supports staff training, operational consistency, and communication across teams.

**Objective**

To create a concise, easy to follow workflow guide for clinical staff that improves documentation accuracy, reduces errors, and standardizes communication around EMR usage.

**Approach**

- Interviewed clinical staff to understand current workflow challenges and documentation inconsistencies.
- Mapped the end to end patient intake and documentation process.
-	Identified gaps in data entry, terminology usage, and communication handoffs.
-	Created a structured workflow guide with step by step instructions, terminology definitions, and accuracy checkpoints.

**Outcome**

-	Improved documentation consistency across staff.
-	Reduced EMR errors and rework.
-	Provided a training resource for onboarding new team members.
-	Strengthened communication between clinical and administrative teams.

**Skills Demonstrated**

Clinical documentation • Communication clarity • Workflow mapping • User enablement • Healthcare operations
